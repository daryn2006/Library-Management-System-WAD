Controller example

@PostMapping("/register")
public ResponseEntity<?> register(@Valid @RequestBody User user) {
    userService.register(user);
    return ResponseEntity.ok("Success");
}

Builder / Factory notes

Builder

Use it when a page or response has optional parts, like avatar, role, statistics and quick links.

Factory

Use it when one action needs different output strategies, like file download or an external URL.
